SOP-8 SPV100A4
---------------------------

独立执行音频解码播报功能的最小封装：

 - 可作为从机通过UART/IIC/IR等方式进行软件控制。
 - 具备PWM输出能力，实现LED呼吸灯等效果。
 - 支持ONOFF按键形式的低功耗唤醒。
 - 支持以AD Key形式进行按键扩展。
